Exploring Spatial Relationships with CSS
So there's this thing called the CSS Box Model. Maybe you've heard of it. It consists of four different things every HTML element has:
  • a margin value
    • what makes space between elements
  • a border value
    • what gives an element a literal border(think thinkness of border)
  • a padding value
    • what fills up an element with extra space
  • the content itself of the element
In addition to those things, every element also has a display style. This changes the way the elements 'stack' next and around each other. In fact, it's how this box doesn't stretch all across the screen.
